ABOUT

I'm a software engineer working in distributed systems, networking & virtualization. My start in cloud was at AWS on EC2. I developed and owned the control plane hypervisor agent from scratch to GA. Work at EC2 expanded to the conception of VPC. I took ownership of the development & delivery of VPC's control plane, user & internal APIs.

After VPC GA I joined the nascent Amazon Aurora project. I developed the storage RPC stack, IO and task scheduling. Later I delivered Virtual Machines for Oracle Cloud Infrastructure. At OCI I designed & patented novel operations for OCC based database systems. These server managed operations elinated a significant & common class commit conflicts.

Recently I've been working in B2B on an EDI platform. In the past I've worked in EFT implementing transaction processing platforms. I implemented 10K connection support for Java (before non-blocking IO support in the JDK). Additionally I implemented EMV support and HSM vendor independent cryptographic libraries.
